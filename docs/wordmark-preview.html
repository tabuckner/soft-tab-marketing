Notes

What to Look For

Each option shows the wordmark in two treatments: solid color and gradient text. Both are viable — the gradient version adds brand color but may not work at very small sizes.

The nav simulation shows how each font feels in context — as a logo alongside navigation links and a CTA button.

Pay attention to:

  • How the hyphen looks in each font (some handle it better)
  • How "TAB" in all-caps reads alongside lowercase "Soft"
  • Whether the font feels technical enough for an engineering consultancy
  • Which one you'd be proud to see at soft-tab.com

All four fonts are free for commercial use. The logo icon shown is your actual Soft-TAB logo.
