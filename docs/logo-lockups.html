4. Clear Space & Minimum Sizes

Minimum clear space around the logo equals the height of the icon (1x). Never crowd the logo.

1x 1x 1x 1x Soft-TAB

Minimum Sizes

Horizontal lockup: min 120px wide
Stacked lockup: min 80px wide
Icon only: min 16px (favicon), 32px (UI)

Clear Space Rule

Minimum padding around any lockup = 1x the icon height. In the horizontal lockup with a 40px icon, that's 40px of clear space on all sides.

Icon Border Radius

Scales with size: ~20% of icon width.
32px = 7px radius, 48px = 10px,
64px = 14px, 96px = 20px.
